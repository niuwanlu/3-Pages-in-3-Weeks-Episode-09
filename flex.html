Flexible Box

Aims at providing a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown and/or dynamic

Basics & Terminology

Flexbox is a whole set of properties not a signle property.

flex-basis

Value of auto is definded by width of element or width of element content

flex-grow

Notice: Default 0 should not be growed. If give an positive，width of element will grow according to scale to take up available space of main axis.

flex-shrink

Notice: Default 1 should be shrinked. If give an position, width of element will shrink according to scale.

Hypertext

Markup Language

HTML history

Physicist Tim Berners-Lee

1980 - prototype ENQUIRE

1989 - propose an Internet-based hypertext system

1990 - write the browser and web server

1991 - first public description

1993 - Internet Engineering Task Force - first proposal for an HTML specification
